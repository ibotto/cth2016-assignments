ALL LEVELS EASY READER: A PROTOTYPE


Description: This program searches user-entered text, locating adjectives and adverbs, in order to substitute the found adj and adv with synonyms retrieved from a Thesaurus API.  

Modules program makes use of:
    - Parts of Speech (pos) 
    - Chance
    - Express 


Link to API: https://words.bighugelabs.com/api.php 

authors: ibotto and jdigirolamo
